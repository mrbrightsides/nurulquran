# Contributing to Nur Al-Quran

We welcome contributions from the community! Whether you're fixing a bug, adding a feature, or improving documentation, your help is appreciated.

## How to Contribute

1. **Fork the Repository**: Create your own fork of the [nurulquran](https://github.com/mrbrightsides/nurulquran) repository.
2. **Create a Branch**: Create a feature branch for your changes (`git checkout -b feature/amazing-feature`).
3. **Make Changes**: Implement your changes, ensuring you follow the existing code style and include types.
4. **Lint & Test**: Run `npm run lint` to check for any TypeScript errors.
5. **Commit**: Write clear, descriptive commit messages.
6. **Push**: Push your changes to your fork.
7. **Pull Request**: Open a Pull Request against the `main` branch of the original repository.

## Code Style

- Use **TypeScript** for all logic.
- Use **Tailwind CSS** for styling.
- Follow functional component patterns with React hooks.
- Ensure all new components are responsive and accessible.
